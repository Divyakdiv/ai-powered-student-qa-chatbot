# AI-Powered Student Q&A Chatbot

An AI-powered chatbot that helps students get instant answers to subject-related questions, built with a Node.js/Express backend and a vanilla JS frontend, powered by the Groq AI API.

## Features

- Ask any academic question and get an instant AI-generated answer
- Subject selector with custom "Other" subject support
- Markdown-rendered responses (bold text, lists, code blocks)
- Clean, responsive UI

## Tech Stack

- **Frontend:** HTML, CSS, JavaScript
- **Backend:** Node.js, Express
- **AI:** Groq API (Llama 3.1)
